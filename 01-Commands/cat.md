# cat

## What it does
Displays the contents of a file.

## Basic syntax
```bash
cat file
```

## Common usage
```bash
cat notes.txt
cat file1.txt file2.txt
```

## Important flags
- -n → show line numbers
- -b → number non-empty lines

## DevOps use cases
- Reading configuration files
- Quickly checking small log files

## Notes / Gotchas
- Not suitable for very large files
- Use less or tail for logs
