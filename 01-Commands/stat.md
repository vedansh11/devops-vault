# stat

## What it does
Displays detailed information about a file or directory.

## Basic syntax
```bash
stat file
```

## Common usage
```bash
stat app.log
stat /etc/passwd
```

## DevOps use cases
- Checking file permissions and ownership
- Viewing inode information
- Debugging permission problems

## Notes / Gotchas
- Shows access, modify, and change times
- Useful for forensic debugging
