# ls

## What it does
Lists files and directories.

## Basic syntax
```bash
ls [options] [path]
```

## Common usage
```bash
ls
ls -l
ls -a
ls -lh
```

## Important flags
- -l → long listing
- -a → show hidden files
- -h → human readable sizes

## DevOps use cases
- Checking files on servers
- Verifying deployments

## Notes / Gotchas
- Hidden files start with .
